# 多模态面试评测平台

## 项目介绍

本项目是一套基于大语言模型（LLM）和多模态技术的智能模拟面试评测平台。

支持语音、视频、文本交互，实现 AI 面试官、智能出题、多维度评测以及面试报告生成。


## 技术栈

### 后端

- Spring Boot
- MyBatis Plus
- MySQL
- Redis
- MinIO


### AI

- LLM
- LangChain4j
- RAG
- AI Agent


### 多模态

- ASR 语音识别
- TTS 语音合成
- OpenCV 视频分析


## 项目模块

- 用户中心
- 简历解析
- AI 出题
- 模拟面试
- 智能评测
- 面试报告
- 后台管理
