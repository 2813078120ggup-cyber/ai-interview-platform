# 多模态智能模拟面试评测平台

> 基于大语言模型（LLM）和多模态人工智能技术的智能模拟面试评测平台，支持文本、语音、视频多种交互方式，实现 AI 面试官、智能出题、多维度评测以及面试报告生成。

---

# 📖 项目简介

随着人工智能技术的发展，传统面试训练方式存在面试成本高、反馈不及时、评价标准不统一等问题。

本项目设计并实现一套基于大语言模型（LLM）、语音识别（ASR）、语音合成（TTS）以及多模态分析技术的智能模拟面试评测平台。

平台通过构建 AI 面试官，实现从面试准备、智能出题、实时交互、回答分析到综合评价的完整流程，为用户提供接近真实招聘场景的模拟面试体验。

主要功能：

- AI 智能生成面试问题
- AI 模拟面试官
- 文本、语音、视频多模态交互
- 智能追问
- 自动化面试评分
- 面试报告生成
- 个性化能力分析

# 📌 开发规范约定

## Git分支规范

采用 Git Flow 开发模式：

```
main

  ↓

develop

  ↓

feature-xxx
```


分支说明：

| 分支      | 说明         |
| --------- | ------------ |
| main      | 稳定发布版本 |
| develop   | 开发集成版本 |
| feature-* | 功能开发分支 |
| bugfix-*  | Bug修复分支  |


---

## Commit提交规范

格式：

```
type: description
```


示例：

```
feat: 完成用户登录模块

fix: 修复JWT认证异常

docs: 更新项目文档

refactor: 优化业务代码
```


提交类型：

| 类型     | 说明     |
| -------- | -------- |
| feat     | 新功能   |
| fix      | Bug修复  |
| docs     | 文档修改 |
| refactor | 代码重构 |
| test     | 测试代码 |
| chore    | 工程配置 |
