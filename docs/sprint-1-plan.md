# Sprint-1 开发计划

## 1. Sprint概述

| 项目 | 内容 |
|---|---|
| Sprint | Sprint-1 |
| 周期 | 2周 |
| 项目 | 多模态智能模拟面试评测平台 |
| 目标 | 完成系统基础框架搭建，实现用户体系、题库管理、面试预约与基础面试流程闭环 |

---

# 2. Sprint目标

本次 Sprint 主要完成平台 MVP 版本基础功能开发：

- 用户注册登录
- JWT身份认证
- 角色权限控制
- 题库管理
- 创建面试
- 面试流程控制
- 提交面试答案
- AI评测接口预留

形成最小业务闭环：

```
用户登录

↓

HR创建面试

↓

面试官开始面试

↓

候选人回答问题

↓

结束面试

↓

进入AI评测流程
```
